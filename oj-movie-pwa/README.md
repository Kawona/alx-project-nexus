This is my final project for the ALX Software Engineering program.
OJ Movies is a simple web application where you can explore movies, check details, save movies to your watchlist, and watch trailers. I built it using React and The Movie Database (TMDB) API. This project helped me practice everything I learned in frontend development.



## Features

- Browse popular, top-rated, and now-playing movies

- Search for any movie using the search bar

- View full movie details (poster, description, actors, ratings, etc.)

- Watch trailers (YouTube)

- Add movies to your personal watchlist

- Remove movies from watchlist

- Fully responsive UI

- Hosted online so anyone can use it
